
# 📦 AI Video Object Detection Project

## 🚀 Overview

This project implements a **real-time video object detection system** using deep learning models. It detects and classifies objects in video streams with high accuracy and optimized performance.

---

## 🧠 Features

* Real-time object detection from video
* High accuracy using YOLO models
* Supports video file & webcam input
* Optimized for speed and performance
* Bounding box visualization with labels

---

## 🛠️ Tech Stack

* Python
* OpenCV
* YOLOv5 / YOLOv11/ VGG / Autoencoders
* NumPy
* PyTorch

---

## 📊 Performance

* Achieved **~93% mAP (Mean Average Precision)**
* Reduced inference time by **~25%**
* Real-time detection with optimized FPS

## ▶️ How to Run

### 1. Clone Repository

```
git clone https://github.com/prassu02/AI-Video-Object-Detection_Project.git
cd AI-Video-Object-Detection_Project
```

### 2. Install Dependencies

```
pip install -r requirements.txt
```

### 3. Run Project

```
python main.py
```
